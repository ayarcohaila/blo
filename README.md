# blo
Static blog generator.

## Features
 * Simple and configurable though JSON

## Installation

     python setup.py install

## Usage
Create a new blog:

     blo create myblog
     cd myblog

Create a new page orr post:

     blo post hello-world

Build blog:

     blo build

Run blog locally:

     blo serve

## Architecture
 * Blog config
 * Page templates
 * Page plugin generator
